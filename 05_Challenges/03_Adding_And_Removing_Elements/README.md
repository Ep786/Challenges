#   Project: Adding And Removing Elements

### Description
Learn how to modify a set in Python by adding and removing elements.

### Pseudocode

    Start
    Create a set called 'numbers' with {1, 2, 3, 4, 5}.
    Add 6 to the set.
    Add 7 to the set.
    Remove 2 from the set.
    Show the updated set.
    End
