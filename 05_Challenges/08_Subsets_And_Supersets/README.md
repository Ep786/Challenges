#   Project: Subsets And SuperSets

### Description

Work with subsets and supersets in Python.

### Pseudocode

    Start
    Create set_a with {1, 2, 3, 4, 5}.
    Create set_b with {2, 3, 4}.
    Check if set_b is a subset of set_a.
    Print the result.
    Check if set_a is a superset of set_b.
    Print the result.
    End
